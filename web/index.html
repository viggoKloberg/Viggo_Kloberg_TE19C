Info om webprogrammeringsbegrepp

Element:

Ett element är, I Html, en full tag och dess inenehåll

HTML:

HTML betyder "hypertext markup language, och är ett programmerinsspråk för webbprogrammering

ID:

ID ges till ett element, så att man kan länka till den. Måste vara unik för varje element

Class:

Class ges också till element, för att ge dem namn

Meta:

Meta är en tag som används för datainformation om datan isej

Responsive:

Responsive design är när websidan automatiskt ändrar sig själv, för att passa på alla enheter

Http:

Http står för "hypertext transfer protocol". Används för att transpotera data över webben

Https:

Https är samma som http, fast krypterat för ökad säkerhet

Compressed data:

Compressed data är data som compresserats ihop för att ta mindre plats

CDN:

CDN står för "content delivery network". Det är flera utspridda servrar som arbetar tillsammans för att ge snabbare uppkoppling

Local Bootstrap:

Local Bootstrap är en metod...

JQuery:

JQuery är ett hjälpmedel för javascript, som förenklar annars komplicerade uppgifter

Flexbox:

Flexbox är ett hjälpmedel för att ändra position på items

Grid:

Grid är osynliga strukturer som kombinerar ihop alla element i en webbsida

Code snippets:

Code snippets är små biar av kod du kan klistra in, för olika användningar
